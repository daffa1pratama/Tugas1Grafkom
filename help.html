Menu Help

How to Draw

  1. Pilih mode Create
  2. Pilih model objek
  3. Pilih warna dari objek
  4. Gambar!
    • Line : klik 2 (dua) titik pada canvas
    • Square, Hexagon, Octagon : pilih ukuran objek dan klik pada canvas
    • Polygon : pilih banyaknya sisi polygon dan klik titik pembentuk polygon pada canvas

How to Move Square, Octagon, Hexagon, Polygon

  1. Pilih mode Move
  2. Pilih model objek yang ingin dipindahkan
  3. Klik pada objek yang akan dipindah
  4. Jika muncul tulisan An object has selected, berarti objek telah dipilih
  5. Klik pada titik lain pada canvas untuk memindahkan

How to Resize/Move Line

  1. Pilih mode Move
  2. Pilih model objek Line
  3. Klik pada salah satu titik pada garis
  4. Klik pada titik lain untuk mengubah panjang garis

How to Resize Square

  1. Pilih mode Resize
  2. Pilih model objek Square
  3. Klik pada objek yang akan diubah ukurannya
  4. Jika muncul tulisan An object has selected, berarti objek telah dipilih
  5. Ganti ukuran objek menggunakan slider Size
  6. Klik canvas untuk melakukan unselect objek

How to Update Color

  1. Pilih mode Update Color
  2. Pilih warna menggunakan color picker
  3. Pilih model objek yang ingin diubah warnanya
  4. Klik pada objek tersebut

How to Save File

  1. Isi nama file pada kolom yang disediakan
  2. Klik SAVE untuk menyimpan

How to Load Save File

  1. Klik Choose File
  2. Pilih file penyimpanan yang akan dibuka
